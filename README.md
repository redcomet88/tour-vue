## 1 通过这个系列教程可以学习到什么？

- ✅  从0开始搭建一个  vue + flask 的数据分析系统；
- ✅  了解系统的整体架构，前后端交互、 后端与数据库的交互方式；
- ✅  几种常用的 echarts 图形的如何使用；
- ✅  element-ui 搭建管理程序布局风格的系统；
- ✅  flask 后端框架作为服务端使用；
- ✅   推荐算法的集成（如果时间安排来得及的话）；
